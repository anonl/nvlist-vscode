
# NVList Visual Studio Code extension

## Features

- Syntax highlighting for .lvn files
- Word count in status bar (.lvn files)
- Autocomplete
-- Image paths (in string constants)
-- Speaker names (at start of line)
- Debug adapter
-- Start/stop
-- Threads
-- Stacktraces
-- Breakpoints
-- Step in/out/over
-- Source lookup
-- Hover (global variables only for now)
-- REPL
